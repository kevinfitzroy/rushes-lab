方案 A:Cyberduck(GUI)

1. 下载 Cyberduck

下载 macOS 下载 Windows 官网 / 其它版本

2. 下载我们的连接 profile(预填 endpoint + region)

下载 poc-minio.cyberduckprofile

双击 .cyberduckprofile 文件,Cyberduck 自动导入 profile。

3. 新建连接

Cyberduck 顶部 + 按钮 → 选 Rushes Lab PoC MinIO(从 profile 导入的)→ 输入 access key + secret key → 连接。

4. 续传 / 大文件

Cyberduck 上传大文件时自动 multipart;断网或关闭后重连,会从中断 part 续传(右键传输列表 → Resume)。

⚠️ 第一次连接:Cyberduck 可能弹 "Disk profile" 安全确认,选 Allow。

方案 B:rclone(CLI)

1. 安装 rclone

# macOS (Homebrew)
brew install rclone

# Windows (Scoop / Chocolatey)
scoop install rclone
# 或:choco install rclone

# 直装(任一系统):https://rclone.org/downloads/

2. 下载 config snippet

下载 rclone-poc.conf

拷贝其中 [poc-minio] 段到 rclone 配置文件:

  • macOS / Linux:~/.config/rclone/rclone.conf
  • Windows:%APPDATA%\rclone\rclone.conf

或直接命令:rclone config(交互式)。

3. 常用命令

# 列 bucket
rclone ls poc-minio:

# 上传大文件(进度条 + 自动 multipart + 续传)
rclone copy ~/Movies/big.mp4 poc-minio:incoming/alice/ --progress

# 下载
rclone copy poc-minio:incoming/alice/big.mp4 ./ --progress

# 同步整个目录(双向 sync 用 bisync,单向用 sync/copy)
rclone copy ~/material poc-minio:incoming/alice/material --progress --transfers=4

# 断网恢复:rclone 自动检测已上传 part,重连后续传(默认 retry 3 次)
# 强行 resume:--ignore-existing 跳过已有,--immutable 标已传部分

4. 挂载为本地盘(可选)

# macOS(先 brew install macfuse,且系统设置允许 kext)
rclone mount poc-minio:incoming /Volumes/poc-incoming --vfs-cache-mode writes &

# Windows(先装 WinFsp:https://winfsp.dev/)
rclone mount poc-minio:incoming X: --vfs-cache-mode writes

方案 C:MinIO 官方 CLI (mc) — 最纯

# 安装(任一)
brew install minio/stable/mc          # macOS
scoop install mc                      # Windows
# 或直装 https://min.io/docs/minio/linux/reference/minio-mc.html

# 配 alias
mc alias set poc https://rusheslab.taoxiplan.com alice alicesecret-poc-2026-32chars-pad

# 命令
mc ls poc/
mc cp big.mp4 poc/incoming/alice/
mc cp poc/incoming/alice/big.mp4 ./
# mc 不天然有 progress + 续传 UX,大文件推荐用 rclone 或 Cyberduck

常见问题

SSL 证书警告?

rusheslab.taoxiplan.com 是 Let's Encrypt 真证书,**不应有警告**。如有(可能旧浏览器/客户端 CA bundle 老),Cyberduck 可点 "Allow once",rclone 加 --no-check-certificate(不推荐生产)。

权限 403?

bob 只能写 incoming/bob/,不能写 incoming/alice/ / private-*/。alice 全权。

上传卡死?

rclone:--low-level-retries=10 --retries=5 多重试;Cyberduck:右键传输 → Resume。MinIO 端 multipart 不完成不入 bucket,失败可 abort 清理。
