# material-storage 操作手册

欢迎使用 material-storage(下称"素材库")。这里是给**业务用户**(剪辑师、摄影师、项目管理员)的一份上手指南,不涉及任何代码或运维细节。

> 这份文档跟"权限模型"一起放在 **demo-onboarding** 演示项目里。任何登录用户都能看到这个项目和它的内容。

---

## 1. 登录

打开 `https://rusheslab.taoxiplan.com/ms-static/web`(或飞书工作台里的 material-storage 入口),浏览器会自动跳到飞书 OIDC 授权页。授权后会跳回素材库的项目列表页。

**已知坑**:不要走 `http://8.156.34.238/...` 这个 IP 入口 — OAuth 回调固定在域名上,IP 入口会"state mismatch"登录失败。README 已标注。

---

## 2. 三种身份,先认清自己

| 身份 | 含义 | 一般是谁 |
|---|---|---|
| **系统管理员** | 能创建项目、管理全局审计;头像旁会有 `SYSTEM ADMIN` 金标 | 只有少数几个人 |
| **项目管理员** | 在某个项目内能改成员/权限/邀请别人进敏感文件夹 | 项目的发起人或被指派的负责人 |
| **普通成员** | 浏览/上传/下载;权限取决于项目内被赋予的角色 | 大多数日常使用者 |

在右上角点头像 → "我的权限"(`/my-permissions`)能看到自己在每个项目里的角色清单。

---

## 3. 项目列表页

进系统后第一眼就是项目列表。每张卡片:

- **项目名 + code**(英文短名,如 `q2-launch`)
- **可见性 tag**:`public` 所有人都看得到 / `private` 仅成员可见
- **创建时间**、**当前 user 的角色 badge**(如果你是这个项目成员)

点击卡片进入项目详情页。

---

## 4. 项目详情页(三栏 workspace)

| 栏 | 内容 |
|---|---|
| **左栏 (260px)** | 文件夹树。普通文件夹有 📁 图标,敏感文件夹有 🔒 图标 |
| **中栏** | 当前选中文件夹的文件列表(支持多选 + 批量删除) |
| **右栏 (320px)** | 选中单个文件后的详情面板:大小/类型/上传者/缩略图/操作按钮 |
| **顶部** | ActionsBar:新建文件夹、上传文件、邀请成员 |

### 4.1 新建普通文件夹

- 点顶部 "新建文件夹"
- 输入名字(如 `02-工作素材` / `02a-面诊与术前`)
- 可选:勾选 "父级文件夹"(嵌套)
- ⚠️ **不要**勾"标为敏感"除非你真要建敏感文件夹

### 4.2 上传文件

- 拖拽文件到中栏,或点 "上传"
- 大文件会自动走分片上传(支持 500MB+ 视频,断点续传)
- 上传完毕后图片自动生成缩略图(几秒内)

### 4.3 下载

- 单文件:中栏列表点行右侧 ↓ 或右栏 "下载"
- 系统返回一个短期 presigned URL,有效 15 分钟

### 4.4 分享单文件给别人

- 在右栏点 "分享" → 生成短链(`/s/<token>`)
- 短链默认需要对方登录素材库才能打开
- 短链同时推送一张飞书 IM 卡片给被分享人(若已配置飞书 app)

## 5. 敏感文件夹

敏感文件夹(🔒)有几个**重要**区别:

1. **只能挂在项目根目录下**,不能嵌套在普通文件夹里
2. **下面不能再建子文件夹**(限一级)
3. **不继承项目角色** — 即使你是项目管理员,**也看不到**任何敏感文件夹里的东西,除非你被**显式邀请**进去
   - **例外**:你自己**新建的**敏感文件夹会自动给你 viewer 权限(否则会进入"自己建完自己看不到"死循环)
4. 想给别人访问 → 项目管理员点敏感文件夹 → "邀请成员" → 选用户/部门/用户组 + 给 viewer 或 downloader 等级

详见 **权限模型.md** §3。

---

## 6. 想访问没权限的资源,怎么办

打开 demo-onboarding 项目里的"03-敏感示例"文件夹试试,你会看到一个 `RequestAccessModal` 弹窗 — 提交一份**访问申请**:

- 填申请理由(必填,审计留痕)
- 申请时长(默认 24 小时,可选永久)
- 提交后:项目管理员会收到飞书 IM 卡片,在卡片上一键批准/拒绝

申请状态在 "我的权限" → "临时授权" 段能看到。

## 7. 顶部右上角的快捷入口

| 元素 | 干嘛 |
|---|---|
| `Ctrl+K` 命令栏 | 快速跳转任意项目/文件夹 |
| "我的申请" 菜单 | 看自己提交/收到的申请(`/approvals`) |
| 头像下拉 → "退出登录" | 清 session,回到飞书 OIDC 起点 |

## 8. 已知体验细节(field tester 视角)

- 飞书 PC webview 内嵌打开素材库,体验跟系统 Chrome 浏览器基本一致;少数边界 case(如缩略图刷新、退出登录跳转)历史上出过差异,如遇异常以浏览器复测为准
- 缩略图首次进入文件夹有可能 race condition;F5 刷新即可
- 上传过程中的进度可视化在右下角"任务中心"浮动按钮里
- 错误提示如果看到"会话已过期",刷新页面后会重新 OIDC
